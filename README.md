# Statistical Analysis & A/B Testing Project

## Опис проєкту
У цьому проєкті зібрано кілька кейсів із прикладними задачами статистики та аналізу даних.  

**Мета:** показати практичне застосування класичних статистичних тестів (Z-test, T-test, χ²-test) та методології A/B тестування на реальних бізнес-кейсах.

## Використані інструменти
- **Python** (numpy, pandas, scipy, statsmodels, matplotlib, seaborn)
- **Jupyter Notebook** для інтерактивного аналізу
- Статистичні методи: Z-test, T-test, χ²-test, пропорційні тести, довірчі інтервали
- Візуалізація даних для інтерпретації результатів

## Структура ноутбука
1. **Аналіз великих вибірок (Z-test)**  
   - Перевірка середнього чеку за липень 2025 року  
   - Виконання тесту вручну та за допомогою **statsmodels**  
   - Інтерпретація результатів  

2. **Тести для малих вибірок та пропорцій (T-test)**  
   - Аналіз часу до покупки після редизайну сайту (мала вибірка)  
   - Перевірка впізнаваності логотипу після рекламної кампанії  
   - Використання t-test та пропорційного z-test  
   - Побудова довірчих інтервалів та висновки  

3. **Комплексний аналіз A/B тесту: Cookie Cats**  
   - Планування експерименту (оцінка потрібної кількості користувачів)  
   - Порівняння утримання гравців (retention_1 та retention_7) між групами  
   - Виконання z-test та χ²-тесту  
   - Аналіз довірчих інтервалів та інтерпретація результатів  

## Результати
- Продемонстровано застосування статистичних тестів для бізнес-задач.  
- Побудовано кейси, які показують, як дані перетворюються на інсайти для прийняття рішень.
